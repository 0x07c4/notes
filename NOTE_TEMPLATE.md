## Metadata

| Field | Value |
| --- | --- |
| Language | `zh-CN` |
| Type | `note-type-here` |
| Status | `draft` / `active` / `evergreen` / `archived` |
| Summary | 用一句话说明这篇笔记要解决什么问题。 |
| Topics | `topic-1`, `topic-2`, `topic-3` |
| Keywords | `keyword-1`, `keyword-2`, `keyword-3` |
| Related | [related-file.md](./related-file.md) |
| Last Updated | `YYYY-MM-DD` |

## 一句话判断

- 先写结论，不要先写背景。

## 核心内容

### 问题

- 这篇笔记要回答什么。

### 判断

- 你的主要判断是什么。

### 影响

- 对架构、产品、检索或后续决策有什么影响。
